# All CGL dependencies

> [!CAUTION]
> All of the following have to be downloaded.  
> We aim to have all dependencies on vcpkg, apt or both.  
> All of these dependencies can also be downloaded manually from their website.  

---

- [GLFW](https://www.glfw.org) Is found on [vcpkg](https://vcpkg.io) and (Advanced Package Tool) apt.
- [GLAD](https://glad.dav1d.de) Is found on [vcpkg](https://vcpkg.io) and [GLAD's downloading website](https://glad.dav1d.de).
- [OpenAL](https://www.openal.org) Is found on [vcpkg](https://vcpkg.io) and (Advanced Package Tool) apt.
- [NanoVG](https://github.com/memononen/nanovg) Is found on [vcpkg](https://vpckg.io) and their [Github Page](https://github.com/memononen/nanovg).
- [LibVorbis](https://github.com/xiph/vorbis) Is found on [vcpkg](https://vpckg.io) and (Advanced Package Tool) apt.


> [!TIP] 
> All of the current libraries are on vcpkg  
> So if you download vcpkg you can then  
> ```vcpkg install packagename``` on all the packages and after that  
> ```vcpkg integrate install``` to integrate them in to your visual studio project automatically.
